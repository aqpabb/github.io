Comandos para iniciar repositorio:
Configuração:

git config --global user.name "aqpabb"
git config --global user.email "aqpa.bb@gmail.com"

# De dentro da pasta do projeto
git init
git status
git add .
git status
git commit -m "primeiro commit"
git remote add origin https://github.com/aqpabb/github.io
git put -u origin master


Comandos para atualizar arquivo:

Faz Upload somente de index.html
Usar CTRL+C  CTRL+V

function atual {
git add index.html
git commit -m "commit atual"
git push
git status
};atual;


Faz Upload de todos os arquivos
Usar CTRL+C  CTRL+V

function atual {
git add .
git commit -m "commit atual"
git push
git status
};atual;
